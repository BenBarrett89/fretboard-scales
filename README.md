# fretboard-scales #

> A fretboard scale display application while trying to learn [Nunjucks](https://mozilla.github.io/nunjucks/)

## Running ##

### Prerequisites ###

* Node.js
* `npm`

### Installing dependencies ###

```console
npm install
```

### Running ###

```console
npm start
```

### Accessing ###

[http://localhost:5000/](http://localhost:5000/)
